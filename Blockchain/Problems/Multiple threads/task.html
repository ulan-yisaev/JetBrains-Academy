Here's a class that extends the Thread class.

class WorkerThread extends Thread {
    
    @Override
    public void run() {
        // the method does something
    }
}

1) Create two instances of the given class and set the names like "worker-X", where X is any suffix (the method setName(...)).

2) Start created threads. The method run of each instance must be executed in a new thread.
